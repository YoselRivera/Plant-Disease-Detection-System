# Plant-Disease-Detection-System

## Overview

This project was developed collaboratively as part of a university machine learning course. Our team designed and implemented a plant disease detection system using Python and PyTorch to classify plant diseases from leaf images using convolutional neural networks (CNNs).

The project demonstrates the application of computer vision and deep learning techniques to automate disease identification, supporting more accurate and efficient plant health assessment.
